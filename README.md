PocketStudy is a beginner-friendly React app that helps students plan focused study sessions using the Pomodoro technique.

## Features

- Pomodoro timer with start/pause/reset
- Emoji celebration and confetti when a study session completes
- Daily, weekly, monthly target setting and simple chart
- Reminder bell and browser notifications when sessions near completion
- Local storage for targets and settings

1. Ensure Node.js >= 16 is installed.
